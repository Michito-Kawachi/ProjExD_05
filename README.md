# ゲーム のタイトル
## 実行環境の必要条件
* python >= 3.10
* pygame >= 2.1

## ゲームの概要
主人公キャラクターを十字キーなどで行動決定し、戦闘を行う

## ゲームの実装
### 共通基本機能
* プレイヤーの枠
### 担当追加機能
* 敵の攻撃（自分の実装箇所）
* プレイヤーの動き
* プレイヤーの選択と行動(戦う、道具)
* プレイヤーの選択と行動(話す、見逃す)
* ライフ管理(UIの表示)
